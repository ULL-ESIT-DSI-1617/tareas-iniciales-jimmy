Página personal

Publicación de una página personal usando Github Pages

Siguiendo los pasos descritos en la página de Github Pages, podremos publicar nuestra página personal de forma sencilla y rápida, teniendo como únicos requisitos:

  • Tener una cuenta en Github.
  • Tener implementada, o ir implementando en el repositorio que crearás, tu página personal con tus archivos .html, .css y .js.
